# glossary.md — Project terms

**Project:** The Corner Table

- **Cover** — one seated guest (restaurant trade term). Capacity is counted in covers, not tables.
- **Slot** — a 30-minute booking window derived from opening hours.
- **Block** — an owner-created closure of a slot or whole day (holiday, private event).
- **No-show** — a confirmed booking where the party never arrived; status set by the owner.
- **Health strip** — the admin Today-view bar showing bookings, email status, errors, month cost.

> 📝 Teaching note: five terms is plenty. The glossary exists so "cover" never gets "corrected"
> to "customer" by a later session that lacks restaurant context.
